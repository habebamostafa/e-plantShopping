# Paradise Nursery - Houseplant Shopping App

A React-based e-commerce application for buying houseplants with shopping cart functionality.

## Features
- Browse houseplants by category
- Add/remove items from shopping cart
- Dynamic cart counter in header
- Responsive design
- Redux state management

## Tech Stack
- React
- Redux Toolkit
- React Router
- CSS3

## Pages
1. **Home Page** - Landing page with company info
2. **Products Page** - Browse and add plants to cart
3. **Cart Page** - Manage shopping cart items

## Setup
1. Clone repository
2. Run `npm install`
3. Run `npm start`
